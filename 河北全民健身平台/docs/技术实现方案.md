## 3. 评价模型算法

### 3.1 均衡性指标
```python
# 基尼系数
def gini_coefficient(data):
    sorted_data = np.sort(data)
    n = len(data)
    index = np.arange(1, n+1)
    return (2 * np.sum(index * sorted_data)) / (n * np.sum(sorted_data)) - (n + 1) / n
```
