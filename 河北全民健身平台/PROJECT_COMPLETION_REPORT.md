## 🚀 系统运行指南

### 快速启动

```bash
# 1. 初始化数据
python3 scripts/quick_init.py

# 2. 启动后端(新终端)
cd backend
pip install fastapi uvicorn pydantic loguru
python main.py

# 3. 启动前端(新终端)
cd frontend
npm install
npm run dev

# 4. 访问系统
# 前端: http://localhost:5173
# 后端API: http://localhost:8000
# API文档: http://localhost:8000/api/docs
```

### Docker部署

```bash
cd docker
docker-compose up -d
```
